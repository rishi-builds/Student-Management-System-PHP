Project Name: Student Management System

Technology Used:
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

Project Description:
This is a Student Management System developed using PHP and MySQL.
The project allows admin/users to perform CRUD operations such as
adding, viewing, editing, and deleting student records.
It also includes a login and logout system for security.

Main Features:
- User Login & Logout
- Add Student Details
- View Student List
- Edit Student Information
- Delete Student Records
- Database connectivity using PHP (db.php)

Project Structure:
- index.php        :  Home page
- login.php        : User login
- logout.php       : User logout
- add_student.php  : Add new student
- edit_student.php : Edit student data
- delete_student.php : Delete student
- view_students.php : View all students
- db.php           : Database connection
- database.sql     : Database file
- style.css        : CSS styling
- script.js        : JavaScript functions

How to Run the Project:
1. Install XAMPP server.
2. Copy the project folder into htdocs directory.
3. Open phpMyAdmin and create a database.
4. Import the database.sql file.
5. Update database credentials in db.php file.
6. Open browser and run:
   http://localhost/Student_Management/login.php
